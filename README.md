# Flask Blog Service

간단한 Flask 기반 블로그 서비스 초기 프로젝트입니다.  
이 프로젝트는 **Git feature branch workflow**를 연습하고, Flask의 기본 API endpoint를 구현하는 것을 목표로 합니다.

---

## 1. Project Goals

- Flask 프로젝트 초기 구조 만들기
- Git 브랜치 전략 실습
- `feature/home` 브랜치에서 기능 개발
- `/` 와 `/home` 기본 endpoint 구현
- 기능 병합 후 로컬 테스트
- GitHub 원격 저장소에 push
